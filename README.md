# Calculator Program

A console-based calculator application developed in Python.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Exponentiation
- Square Root
- Sine Function
- Cosine Function
- Tangent Function
- Previous Calculation History

## Concepts Used
- Functions
- Lists
- Loops
- Match-Case
- Exception Handling
- Math Module

## Language Used
- Python
